## Icon 图标

提供了一套常用的图标集合

### 基本用法

为原始的 SVG 图标提供额外的属性

::: demo

```vue
<template>
  <u-icon icon="overall_input_edit" color="#EF5230"></u-icon>
</template>
```

:::

<!-- ### 参数文档 -->

### Icon 属性

| 属性  |       说明       |  类型   | 可选值 | 默认值 |
| :---: | :--------------: | :-----: | :----: | :----: |
| icon  |     图标类名     | string  |   -    |   -    |
| color |     图标颜色     | string  |   -    |   -    |
| right | 图标是否处于右边 | boolean |   -    | false  |
