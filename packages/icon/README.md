# @upfos/icon

## Install

```bash
npm i @upfos/icon
# or
yarn add @upfos/icon
```

## Usage

```vue
<template>
 <u-icon/>
</template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'
import UIcon from '@upfos/icon'

export default defineComponent({
  components: { UIcon },
  setup() {}
})
</script>
```
