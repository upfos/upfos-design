# @upfos/design

## Install

```bash
npm i @upfos/design
# or
yarn add @upfos/design
```

## Usage

```vue
<template>
 <u-design/>
</template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'
import UDesign from '@upfos/design'

export default defineComponent({
  components: { UDesign },
  setup() {}
})
</script>
```
