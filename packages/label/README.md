# @upfos/label

## Install

```bash
npm i @upfos/label
# or
yarn add @upfos/label
```

## Usage

```vue
<template>
 <u-label/>
</template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'
import ULabel from '@upfos/label'

export default defineComponent({
  components: { ULabel },
  setup() {}
})
</script>
```
