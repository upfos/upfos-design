# @upfos/loading

## Install

```bash
npm i @upfos/loading
# or
yarn add @upfos/loading
```

## Usage

```vue
<template>
 <u-loading/>
</template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'
import ULoading from '@upfos/loading'

export default defineComponent({
  components: { ULoading },
  setup() {}
})
</script>
```
